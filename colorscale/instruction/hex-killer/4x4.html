Hex Sudoku Killer 4x4

Mode: Killer
Grid: 4x4

Objective

Fill the 4×4 grid using hex color codes so that:

  • Each row and column contains all 4 unique hex colors once.
  • Cages (outlined groups of cells) must add up to the given sum.
  • No repeating colors within a single cage.

Tips

  • Use the cage totals to eliminate color combinations.
  • Start with cages that contain only one cell or very few possibilities.
  • Remember hex values may be represented in either color or numerical equivalents (e.g. #FF0000 might represent 15).
Goal: Respect both Sudoku placement rules and cage totals!
